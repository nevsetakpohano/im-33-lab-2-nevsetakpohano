# Calculator — Лабораторна робота №2

**Виконала:** Бабаніна Арина

**Група:** ІМ-33

## Опис проекту

Реалізація спрощеного клавішного калькулятора на Python з:
- Обробкою послідовності натискань клавіш
- Відображенням проміжних станів як у реальному калькуляторі
- Набором юніт-тестів, що покривають усі вимоги завдання
- CI/CD пайплайном через GitHub Actions для автоматичного тестування

## Вимоги
- Python 3.9 або вище
- pip (менеджер пакетів Python)

## Інструкція з налаштування середовища та запуску тестів

### 1. Налаштування середовища

```pip install -r requirements.txt```

### 2. Запуск тестів

```python -m pytest tests/ -v```

Або альтернативно:

```python tests/test_calculator.py```

## Запуск програми
### Інтерактивний режим:

```python src/calculator.py```

### Тестовий режим (запуск всіх прикладів з завдання):

```python src/calculator.py --test```

## Тестування
### Юніт-тести
Проект містить набір юніт-тестів, що покривають:

Тести стану калькулятора (test_calculator_state_*)

Тести парсингу вводу (test_parse_input_*)

Тести валідації послідовності (test_validate_sequence_*)

Тести обробки клавіш (test_handle_keypress_*)

Тести обчислень (test_calculate_*)

Інтеграційні тести з файлами (test_process_file_*)

## Запуск всіх тестів

```python -m pytest tests/ -v```

## CI/CD (GitHub Actions)
Налаштовано автоматичне тестування при кожному:

git push до гілок main або master

Створенні Pull Request до цих гілок

Файл конфігурації: ```.github/workflows/ci.yml```
